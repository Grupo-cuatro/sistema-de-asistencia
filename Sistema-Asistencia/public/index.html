Bienvenido al Sistema de Asistencia Escolar

Plataforma diseñada para gestionar estudiantes, controlar accesos, registrar asistencia diaria, consultar historiales, generar reportes y respaldar información de manera rápida, segura y eficiente.

Ingresar al Sistema

👨‍🎓 Gestión de Estudiantes

Registro, actualización y consulta de estudiantes.

🔐 Control de Acceso

Autenticación de usuarios y manejo de roles.

📋 Asistencia Escolar

Registro diario de asistencia, tardanzas, faltas y justificaciones.

📸 Fotografía de Estudiantes

Identificación visual para mejorar el control escolar.

📅 Historial de Asistencia

Consulta y seguimiento histórico de asistencia.

📊 Reportes y Estadísticas

Reportes por estudiante, carrera, ciclo y fecha, exportables a CSV.

🗄️ Base de Datos SQLite

Toda la información se almacena de forma persistente en el servidor.

🕵️ Auditoría del Sistema

Registro de acciones: accesos, registros, modificaciones y eliminaciones.

Objetivo del Sistema

Facilitar el control y seguimiento de la asistencia escolar, permitiendo una gestión eficiente de estudiantes y docentes mediante herramientas tecnológicas modernas.

En el Sprint 3 se incorpora un servidor Node.js con Express y una base de datos SQLite, agregando roles de usuario, reportes estadísticos, exportación de datos y auditoría de acciones, manteniendo el diseño y la organización del Sprint 2.
